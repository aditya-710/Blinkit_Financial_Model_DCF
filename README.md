# Blinkit Financial Model & DCF Valuation (FY2026–FY2031)

A comprehensive financial modeling project on Blinkit (Eternal Ltd.) covering historical analysis, 5-year projections, DCF valuation, sensitivity analysis, ratio analysis, and investment recommendations.

---

## Project Overview

This project evaluates Blinkit's financial performance and estimates its intrinsic value using Discounted Cash Flow (DCF) methodology.

The model covers:

- Historical financial analysis
- 5-year revenue and profit projections
- Operating and net margin assumptions
- DCF valuation
- Sensitivity analysis
- Competitor benchmarking
- Investment recommendation
- Dashboard visualization

## DCF Valuation Model

### Discounted Cash Flow (DCF) Analysis

The DCF model estimates Blinkit's intrinsic value using:

- WACC = 11%
- Terminal Growth Rate = 4%
- Projected Free Cash Flows (FY2027–FY2031)
- Terminal Value Calculation
- Sensitivity Analysis

**Estimated Intrinsic Value: ₹250 per share**

The model suggests that the stock is undervalued relative to the assumed market price of ₹230.
